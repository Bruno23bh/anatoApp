Datos del pago

Estado del pago: El pago cambiará de estado automaticamente a:
IMPAGO cuando alguna cuota se encuentre impaga.
PAGO cuando todas las cuotas estén pagas.
Honorario acordado: Usalo para recordar qué se le va a cobrar al cliente. Ejemplos:
- El 20 %.
- 2 haberes.
Fecha de comunicación: Es la fecha en que se le informó al cliente sobre los honorarios.

Tareas

Fecha de referencia: Podes usarla como fecha de vencimiento.
Reordenar tareas: Podes ordenar las tareas una encima de otra usando el botón

Dependencias

Iuris ofrece un listado de dependencias que se utilizan con los turnos y expedientes. Si necesitas una dependencia adicional, podés agregarla desde acá. Estas dependencias son visibles para todos los usuarios de tu organización.
